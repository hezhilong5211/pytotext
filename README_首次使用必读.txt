================================================================================
                  链接提取工具 - 首次使用必读
================================================================================

欢迎使用链接提取工具！本工具可以自动从Excel中的链接提取标题和作者信息。

================================================================================
⚠️  重要：首次使用前必须安装Playwright浏览器
================================================================================

本工具需要Playwright浏览器支持，用于处理：
  ✅ 今日头条
  ✅ 百度百家号
  ✅ 抖音
  等网站

如果不安装，这些网站将无法正常提取。

================================================================================
📥 安装方法（二选一）
================================================================================

方法1：一键安装（推荐）
--------------------
1. 双击运行【安装Playwright浏览器.bat】
2. 等待安装完成（约2-5分钟）
3. 完成！

方法2：手动安装
--------------------
1. 打开命令行（cmd 或 PowerShell）
2. 输入：playwright install chromium
3. 按回车，等待完成

================================================================================
📖 使用步骤
================================================================================

安装浏览器后：

1. 准备Excel文件
   - Excel中的链接可以是超链接或纯文本链接

2. 双击运行【链接提取工具.exe】

3. 点击"浏览"按钮，选择Excel文件

4. 点击"开始处理"

5. 等待处理完成（可以在日志中查看进度）

6. 完成后，结果会自动保存为新的Excel文件

================================================================================
💡 提示
================================================================================

- 第一次运行可能会比较慢（需要加载浏览器）
- 处理大量链接建议在网络良好时进行
- 某些网站有反爬虫机制，可能偶尔失败
- 失败的链接会在Excel中标黄，可以手动补充

================================================================================
❓ 常见问题
================================================================================

Q: 为什么今日头条都失败？
A: 没有安装Playwright浏览器。请运行【安装Playwright浏览器.bat】

Q: 安装浏览器失败怎么办？
A: 检查网络连接，或尝试手动安装（见上方"方法2"）

Q: 需要多长时间？
A: 取决于链接数量，平均每个链接1-3秒

Q: 可以中途停止吗？
A: 可以，点击"停止"按钮，已处理的结果会保存

Q: 结果在哪里？
A: 默认保存在原Excel文件同一目录，文件名包含"提取结果"
